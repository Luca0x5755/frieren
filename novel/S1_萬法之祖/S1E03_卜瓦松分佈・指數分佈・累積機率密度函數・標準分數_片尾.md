夜深的港邊小酒館，燈塔守衛和費倫在角落分一壺薑湯。芙莉蓮坐在窗邊，半閉著眼，看不出有沒有在聽。

「姑娘，妳今晚那兩條曲線——」守衛說，「左邊是 Poisson，右邊是 Exponential，對吧？」

「對。」費倫點頭，「次數和間隔。互為對偶。」

「那 PDF 跟 CDF 呢？」守衛接著問，「我聽妳剛剛唸了一次，一個是密度，一個是累積。」

「對。」費倫說，「PDF 是密度——某個位置有多少機會。CDF 是 PDF 的家——把 PDF 從左到右積起來，是『到那個位置為止累積多少』。」

「⋯⋯那 CDF 是 PDF 反過來嗎？」

費倫一頓，差點點頭。窗邊的芙莉蓮這時不睜眼，淡淡接一句：

「⋯⋯PDF 在裡，CDF 在外。」

「啊？」守衛愣住。

「不是反過來。」芙莉蓮說，「是內外。CDF 把 PDF 從一端積到另一端。」

費倫的耳朵微紅——她剛才差點就答錯了。

「⋯⋯Cumulative Distribution Function 是 Probability Density Function 的積分。」她重複一次，把這句話刻進腦裡。

「然後 Z-Score——」守衛繼續問，「妳今晚算給我的那個 Z 等於三。」

「對。」費倫說，「(X 減 μ) 除以 σ。Z 大於等於三，通常判定為異常。」

「那今晚我按警報是對的吧？」

「對。」

守衛笑了笑：「我這輩子第一次被一條公式幫忙做決定。」

「沒事。」費倫笑著回他，「⋯⋯不會是最後一次。」

窗邊的芙莉蓮沒有回頭，把手裡的杯子貼著窗玻璃。窗外那條被一頁筆記吹過的風，往南，還在吹。
