# S2E19｜〈推理行動〉／〈代理者〉／〈工具〉／〈上下文協定〉／〈LLM 運維〉—— ReAct（2022）・AI Agent（2023）・Function Calling（2023）・MCP（2024）・LLMOps（2023）

---

　　四海通旅旅行社。早晨。

　　玄關牆上釘滿彩色圖釘的旅遊地圖，柜檯上擺著四張不同樣式的合約——各家旅館、馬車行、退稅局都有自己的接單格式。老闆娘一臉疲憊，等在門口。

　　「芙莉蓮小姐——這個月我燒掉半年利潤。」

　　「怎麼回事？」費倫問。

　　「上月我換了新 AI 助手——會說話，不會辦事。」老闆娘攤手，「客人要訂三夜加馬車再順便退稅——它只跟人聊旅遊攻略。」

　　柜檯內，一位年輕工程師抱著筆電苦笑。

　　「用的是大模型——會想、會回話——但就是**不動手**。」

　　芙莉蓮翻著地圖打哈欠。

　　「……只會想，不會做。」

---

　　費倫翻開地圖。

　　「每家旅館的接單都不一樣？」

　　「北家先填表、東家先付訂、南家先傳信——格式各自一套。」老闆娘嘆氣。

　　「Token 呢？」費倫追問。

　　「上個月——**燒掉三倍預算**。」工程師壓低聲音，「我不知道錢花去哪。」

　　費倫闔上地圖。

　　「五件事——想、動、接、通、管。」她淡淡，「分五段——前四段我來，最後一段修塔爾克。」

　　修塔爾克在門口愣住，耳根紅了一下。

　　「……喔。」

---

　　費倫先看助手的對話紀錄。

　　「單輪 LLM 只會回答。」她低聲，「讓它——**想一步、做一步、看結果、再想下一步**。」

　　「想？」工程師皺眉。

　　「思考、行動、觀察——迴圈下去，直到事情辦完。」費倫闔眼——心像：一位偵探先想（該查天氣還是訂房？）、再動（查天氣）、再觀察（下雨）、再想（改室內行程）——一步一迴。

　　「……ReAct。」

　　助手換了腦迴路——不再只回話，開始交錯思考與行動。工程師倒吸一口氣。

---

　　老闆娘仍搖頭。

　　「迴圈有了，但它沒記住客人偏好——上次交辦的，全忘了。」

　　「光會迴圈不夠。」費倫答，「要一位完整的**代理者**。」

　　「代理者？」

　　「**四件套**：規劃、記憶、工具、反思。」費倫闔眼——心像：一位管家，先規劃整天行程、記得主人不喝咖啡、會派外傭買菜、晚上回顧哪件沒辦好。

　　「……AI Agent。」

　　助手長成四件齊全的模樣：**Planning ＋ Memory ＋ Tool Use ＋ Reflection**。

---

　　「現在——叫它訂房。」費倫說。

　　「可它怎麼知道我家 API？」工程師問。

　　「**結構化輸出**。」費倫闔眼，「模型不自己上網——它吐出一段規格，讓我們照著打。Anthropic Tool Use、OpenAI Function Calling——同一個魂。」

　　「……Function Calling。」

　　石板上浮出一段 JSON：`{"book_hotel": {"city": "王都北", "nights": 3}}`——旅行社的系統照著打給王都北的旅館。

---

　　老闆娘又皺眉。

　　「我有四家旅館、三家馬車行、兩個退稅局——**每家都要重接一遍？**」

　　「會接到天荒地老。」費倫低聲，「一個協定統一全部。」

　　「統一？」

　　「像 USB-C——插頭一個樣，各家外設都能接。」費倫闔眼，「Anthropic 提出，解決工具整合碎片化。」

　　「……Model Context Protocol。」

　　助手多出一個統一外接口——北旅館、東旅館、馬車行、退稅局，全用同一個插頭。四家一起接、一起收工。

---

　　工程師躍躍欲試。

　　「我自己做一個更簡單的——直接丟給單輪 LLM 訂房！」

　　「等等——」費倫抬頭。

　　他已把客人需求一句丟進舊的單輪大模型——**沒有迴圈、沒有工具呼叫、沒有觀察**。

　　「……LLM。」

　　助手回了一句漂亮的遊記文——**沒訂到房**。空響。

　　「**單輪 LLM 沒有迴圈、也沒有工具。**」芙莉蓮淡淡，「**它只會說不會做。**」

　　「Agent 不是更會講話的 LLM。」費倫接上，「是**會想＋會動＋會接＋會反思**的那一位。」

---

　　老闆娘還沒鬆氣。

　　「上個月 Token 三倍預算——這要怎麼管？」

　　「……我來。」修塔爾克把斧頭拄地，耳根紅但語氣篤定。

　　「戰士也會管模型？」老闆娘愣住。

　　「不是魔法——是運維。」他把一本厚厚的帳冊攤開，「Prompt 哪一版、Token 花多少、回了幾句幻覺、延遲分位——每一欄都要記。」

　　他指著帳本欄位，一欄欄解釋。

　　「這叫——**LLMOps**。」

　　「跟兵站一樣？」修塔爾克自己小聲補，「糧草、工具、回報——每樣都要清點。」

　　費倫笑了一下。

　　「**MLOps 的 LLM 特化版。**」她點頭，「Prompt 版管、Token 成本監控、幻覺偵測——三件是入門。」

---

　　費倫幫老闆娘整理筆記。

　　「ReAct——**思考→行動→觀察**迴圈；與 CoT 差在**加入了工具呼叫**。」她指著石板。

　　「AI Agent——四件套：**Planning、Memory、Tool Use、Reflection**；與單輪對話 LLM 不同。」

　　「Function Calling——**結構化輸出**呼叫外部 API；OpenAI、Anthropic 皆有。」

　　「MCP——Anthropic 的**開放協定**，標準化 LLM 與外部工具／資料源的連接；**AI 的 USB-C**。」

　　「LLMOps——LLM 專用運維；Prompt 版管、Token 成本、幻覺偵測。」

---

　　夕陽西下。

　　石柱後，彩光微微搖晃——辛美爾的身影又浮了出來。

　　「芙莉蓮。」他溫柔，「能想、能動、能花錢、還能接外部——**那就該守規矩了**。」

　　費倫抬頭：「守規矩？」

　　辛美爾笑得溫柔。

　　「**接下來——我們該講法律了。**」

　　彩光散去。芙莉蓮闔上筆記。

　　「……下一章。」她淡淡，「**沙盒、NIST、歐盟 AI 法、台灣草案**——律法全員到齊。」

　　修塔爾克默默把戰斧重新扛起；費倫闔上筆記本；老闆娘在後頭深深一揖。

**考試重點速記**
- 前四項屬 **科目一｜AI Agent**；LLMOps 屬 **科目一｜MLOps**
- **ReAct（Reasoning + Acting, 2022）**
  - **Thought → Action → Observation** 迴圈
  - Agent **代表框架**
  - **與 CoT 差異：加入工具呼叫**
- **AI Agent（2023）**
  - 具**自主規劃、呼叫工具、記憶與反思**能力
  - 核心四件套：**Planning + Memory + Tool Use + Reflection**
  - 與**單輪對話 LLM** 差異要能答
- **Function Calling / Tool Use（2023）**
  - LLM 依**結構化輸出**呼叫外部 API 或函式
  - 代表：**OpenAI Function Calling、Anthropic Tool Use**
  - **Agent 系統基礎**
- **MCP（Model Context Protocol, 2024）**
  - **Anthropic** 提出的**開放協定**
  - 標準化 LLM 與外部工具／資料源連接
  - **解決工具整合碎片化**；稱「**AI 的 USB-C**」
- **LLMOps（2023）**
  - **MLOps 的 LLM 特化版**
  - 三大支柱：**Prompt 版本管理、Token 成本監控、幻覺偵測**

**錯誤示範：** 工程師把 Agent 當單輪 LLM 用——無迴圈、無工具呼叫、無觀察——客人沒訂到房——空響。**單輪 LLM 只會說不會做。**

**本章咒序：** ReAct（想動觀）→ AI Agent（四件套）→ Function Calling（叫工具）→ MCP（統一接口）→ LLMOps（運維清點）。前四段費倫施法，末段修塔爾克以運維之道接棒。
