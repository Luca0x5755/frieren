# S1E02｜〈N 元連語〉／〈詞重之衡〉／〈情感辨析〉—— N-gram（1948）・TF-IDF（1972）・Sentiment Analysis（2002）

## 一、語典塔地下書庫

雪嶺南邊，語典塔立在枯林盡頭。塔身由黑石砌成，每一層都刻滿了古代的字符輪廓。塔外的學者穿著灰長袍，看見三個旅人遠遠走來，便小跑過來迎接。

「芙莉蓮大人，有勞您了。」學者一邊行禮，一邊引他們走下地下石階，「三疊匿名訴狀——我們懷疑是同一人連寫的連環誣告，但沒有證據。」

「……就這個？」芙莉蓮瞥一眼木桌上三疊紙。

「上次同類案件，我們用字頻排了半個月，最後連原告都找不到。」學者苦笑，「這次希望您能教我們——更快的辦法。」

費倫把筆記攤開：「三疊訴狀，要判斷作者是否同一？」

「不只。」學者指著中間那疊，「還要判定情緒——是受害者的憤怒，還是虛構的激動。」

修塔爾克小聲開口：「這個……不會出事吧？萬一判錯人——」

「那就別判錯。」芙莉蓮懶懶說。

---

## 二、常見字的陷阱

費倫先試。

她把三疊訴狀攤開，照字頻由高到低排列，然後讀出前五個最常用字：

「……『的』、『是』、『了』、『也』、『有』。」

學者苦笑，修塔爾克縮縮脖子。

「師父。」費倫抬頭，「這根本看不出哪個詞重要。」

「……嗯。」芙莉蓮把披風拉緊，「常用字頻高，權重被放大。」

「所以是句子沒斷好？」

「不是。是分子沒除。」

費倫皺眉：「分子沒除？」

芙莉蓮沒解釋。

她只對著三疊訴狀輕輕伸出一指。紙面浮字旋轉——「的、是、了」等字先暗下，接著偏用字一個個發亮：「誹謗」「謊言」「證人」「夜半」。她淡淡出聲：

「……Term Frequency–Inverse Document Frequency。」

費倫倒吸一口氣，飛快寫下：

> TF-IDF：以逆文檔頻率壓下常見詞。常見詞**被除掉**權重，不是因為沒斷句。

「這個『逆文檔頻率』是什麼？」

「……在所有文件都常出現的字，權重該低。」芙莉蓮頓了一下，「只在某幾篇出現的字，才是訊號。」

修塔爾克抄下一行，字依舊工整。

---

## 三、連珠短鏈

光亮的偏用字之間，開始有短鏈浮起來。「夜半—闖入」「闖入—撒謊」「撒謊—逃竄」——兩兩連珠成字鏈。芙莉蓮再一揮手：

「……N-gram。」

費倫看得入神：「兩個字兩個字連？」

「二元。三元就三個。」

修塔爾克傻乎乎插話：「那、那四元、五元呢？不是更準？」

「……會死。」芙莉蓮說，「長度越長，涵蓋越窄，抓不到的越遠。」

「抓不到的越遠？」費倫追問。

「N-gram 只看前 N 個字。隔太遠的呼應抓不到。」芙莉蓮懶懶靠上桌沿，「不是假設詞獨立喔——那是 Naive Bayes。別混。」

費倫立刻補上：

> N-gram：僅依固定長度前序詞建模，**無法捕捉長距離依賴**。
> ⚠️「假設詞獨立」是 Naive Bayes，不是 N-gram。

修塔爾克默默把這行字抄進自己的小本子——他最近也開始寫筆記了。

---

## 四、情緒之辨

三疊訴狀的字鏈裡，浮起三種不同顏色的光——紅、青、灰。芙莉蓮最後凝視語氣：

「……Sentiment Analysis。」

學者湊近：「紅色是憤怒嗎？」

「嗯。」

「青是——」

「冷靜。」芙莉蓮指向灰色那疊，「這疊，裝出來的激動。」

學者愣住。

費倫追問：「師父，裝的為什麼會不一樣？」

「……寫法不同。」芙莉蓮指向灰疊，「這疊用的句式、標點、詞彙都像翻譯腔。真的受害者不會這樣寫。」

「所以——情緒判斷會受書寫風格影響？」

「嗯。不同語言、不同文化，書寫風格會造成偏誤。」芙莉蓮打了個哈欠，「叫做隱性偏誤。考題愛考。」

費倫筆記：

> Sentiment Analysis 主要目的：**判斷情感傾向**（正／負／中）。
> ⚠️不同語言／文化書寫風格造成 **Implicit Bias**。

修塔爾克看著灰色那疊發呆：「……連生氣都能裝。」

「人本來就會裝。」芙莉蓮把手收回披風裡，「咒只是把它照出來。」

---

## 五、塔下學徒的空響

正在整理文卷的塔下學徒——一個梳著兩條辮子的小女孩——聽了半天，忍不住探頭：「大人！這個我也會！」

她從桌上抓起一張廢紙，急急寫下幾個詞頻最高的字，嘴裡念：

「『的』最多、『是』次多——所以主題是『的是』——」

筆尖落下，紙面浮起薄薄一層粉紅光暈，才亮了一秒就碎成灰。

修塔爾克嚇得又往後退半步。

「……分子沒除。」芙莉蓮說。

小學徒委屈地摸鼻子：「那個『的』真的最多嘛——」

「所有文件裡都最多。」芙莉蓮沒動，「所以它不重要。」

費倫補上：「她犯的錯跟剛剛我一樣。只看 TF，沒乘 IDF。」

「嗯。」

學者在旁邊記下這段，遞給小學徒一杯熱茶安慰。另一個塔下學徒不服氣，跑過來爭辯：

「那、那我來做情緒——」

他隨手挑了一疊，念出句中「不恨」兩字：「恨——這是憤怒！」

紙面的咒紋才起，立刻爆出一團黑霧，又瞬間消散。

費倫寫得飛快：

> 心像偏差：把「不恨」讀成「恨」，跳過否定字 → 咒紋吐不出正確情緒 → 空響反噬。

芙莉蓮只說一句：「……前面少了一個字。」

學徒呆在原地。

第三名學徒見狀舉手：「N-gram 我會！把 N 調到十——」

「……會抓不到。」芙莉蓮沒讓他動手，「越長越稀疏。什麼都不會亮。」

費倫再補：

> N 越長，涵蓋越稀疏 → 看似精密、其實抓不到遠處呼應。

---

## 六、塔外雪光

離開時已是傍晚。

學者親自送到塔門口，三疊訴狀已被封進木匣。修塔爾克側頭偷瞥費倫那本越寫越厚的筆記——又馬上轉開，耳朵微紅。

「師父。」費倫把行裝繫好，「這三道咒——真的很古老嗎？」

「……嗯。N-gram 是 1948。TF-IDF 是 1972。情感辨析比較年輕——2002。」

「情感辨析最年輕？」

「嗯。沒那麼古。」

修塔爾克怯怯問：「那——這三道都不能單獨用嗎？」

「……N-gram 抓不到遠。TF-IDF 只看詞重。情感只看傾向。」芙莉蓮眯起眼睛，「三把合起來，才讀得出一篇話背後的心思。」

費倫若有所思。

芙莉蓮把披風帽子扣上：「下一站——廢棄神殿。字刻在石頭上，要先撈出來，才能辨名字。」

「OCR 跟 NER？」費倫問。

「……嗯。兩層不一樣。」

「怎麼不一樣？」

「一層是字，一層是意思。」芙莉蓮往前走，「早點睡。」

雪光從塔頂灑下來，把三個人的影子拉得很長。

**一句話口訣**
- **N-gram**：只看前 N 個，抓不到遠處呼應；⚠️「詞獨立」是 Naive Bayes 不是它。
- **TF-IDF**：常見詞被 IDF 除掉權重，不是因為沒斷句。
- **Sentiment Analysis**：判斷情感傾向，不同語言／文化書寫風格造成 Implicit Bias。

**考試重點速記**
- N-gram 不是「假設詞獨立」 → 那是 **Naive Bayes**；N-gram 的真正短板是**無法捕捉長距離依賴**。
- TF-IDF 常見詞權重被放大 → **不要選「缺乏句子邊界」**；答案是 IDF 沒除。
- Sentiment Analysis 主要目的 = 判斷情感傾向（正／負／中），不是找關鍵詞、不是分類作者。
- 書寫風格差異＝**Implicit Bias**（隱性偏誤）。

**心像崩潰對照**
- 省略 IDF、只看 TF → 常用字全暴漲 → 粉紅光暈碎散（小學徒示範）。
- 情緒咒未辨否定字（「不恨」讀成「恨」）→ 黑霧反噬（另一學徒示範）。
- 把 N 調到十、想抓長距離 → N 越長越稀疏，什麼都不會亮（第三學徒示範）。

**下章預告**：廢棄神殿，OCR（字符顯影）／NER（實體命名）——一層是字，一層是意，**兩者不可混**。
