# S1E08｜〈定位羅盤四階〉—— GPS / Wi-Fi / Beacon / NFC（1973–2013）

## 一、山霧岔路

晨霧濃得發白。一條商隊在山腰的岔路口停下，領隊來回踱步——魔法羅盤的指針在胸前亂轉。

領隊見到芙莉蓮一行，立刻鞠躬：「芙莉蓮大人——求您帶我們上山。三日後山頂市集開市，我們的貨還在谷底。」

費倫翻開地圖，上頭三個紅點由山底一路標到山頂：**山脈／山村／市集**。

費倫：「為什麼羅盤不靈？」

「……霧裡靈、出霧不靈；進村靈、出村不靈。」領隊苦笑，「每一段能信的距離都不同。」

修塔爾克摸著戰斧的柄，眼神飄遠了一下：「……範圍。」

費倫回頭：「什麼？」

「每一段——需要的範圍不同。」修塔爾克低聲，「一把羅盤管不了全程。」

芙莉蓮半靠岩石，半瞇著眼：「……該你。」

修塔爾克深吸一口氣。

---

## 二、四句戰場直覺

修塔爾克抽出戰斧——不是要砍，是拿來比劃。

他指向山頂那一片被霧蓋住的輪廓：

「大範圍——靠星。」戰斧一指，「看得到天的時候，連山形都能對上。」

費倫邊聽邊記：「整座山的尺度？」

「嗯。然後——」戰斧略往下收，「進村，靠煙。」

「靠煙？」

「炊煙、雞啼、氣味——這些走不了多遠。」修塔爾克繼續，「進了村，還有更細的——靠手印。」

費倫：「手印？」

「攤位前——每家攤位都有自己的徽紋、招牌。」他戰斧再一指，「站到攤位跟前，徽紋才認得你。」

「最後——」戰斧直直往下一點，「交易——靠碰。」

「碰？」費倫更糊塗。

「錢貨兩清的那一下，手跟手要貼上去。」修塔爾克臉微紅，「不貼，交易不成。」

費倫一字不漏寫下來。

---

## 三、四個方向

領隊一臉困惑：「……修、修師兄，您這四句聽起來像兵法？」

修塔爾克搖頭，把戰斧穩穩立在地上。

「有四個方向。」他說，「我一次指一個。」

第一指——**天**。

「**GPS**。全球定位——靠星。覆蓋最廣、精度最粗。認山不認村。」

第二指——**村口**。

「**Wi-Fi**。靠建築與村落內的訊號。到了村，才認村。」

第三指——**攤位**。

「**Beacon**。靠訊標。走到攤位前幾步，才會被喚起。」

最後一指——**手**。

他把戰斧豎起，向下輕輕一點，點在自己的掌心。

「**NFC**。貼上去才成立。不貼——沒事發生。」

費倫把四個方向對應四個名字寫在筆記上：

> **GPS → Wi-Fi → Beacon → NFC**
> 覆蓋範圍：**大 → 小**

---

## 四、年代別記反

芙莉蓮終於離開岩石，斜眼掃過筆記：「……年代別記反。」

費倫一愣——她剛才順著範圍從大到小寫，下意識覺得年代也是這個順序。

她查了一下標註：GPS 1973、Wi-Fi 1997、**NFC 2004**、**Beacon 2013**。

「……年代跟範圍不一樣。」費倫重新整理，「年代是出現時間；範圍是實際涵蓋。」

修塔爾克補：「考的是**範圍**——行銷精度要看情境。」

「怎麼看情境？」

修塔爾克又豎戰斧。

「要抓一整片山域、城區——**GPS**。」

「要抓一棟商場或整座村——**Wi-Fi**。」

「要抓某一間店、某一個攤位前——**Beacon**。」

「要抓碰到手的那一瞬——**NFC**。」

費倫畫了一張同心圓：外圈 GPS，往內依序 Wi-Fi、Beacon、NFC。

> 易混：**年代**（GPS 1973 → Wi-Fi 1997 → NFC 2004 → Beacon 2013）**與範圍不同向**。

---

## 五、布販的範圍錯配

一行人走到山村市集試行。一位布販攔住修塔爾克，滿臉焦急：

「師兄——我在山腳下擺攤，想給進山的客人推優惠——我裝了 NFC 給他們感應……」

修塔爾克苦笑：「在山腳——客人還沒走到你攤前吧？」

「對啊，他們在山路上完全收不到。」

「NFC 要**碰**才成立。」修塔爾克蹲下，用戰斧點他攤前的一小塊地，「只有這一圈。客人不走到這裡——沒反應。」

布販懊惱：「那、那我該怎麼做？」

「山路上——要**GPS**。」修塔爾克指山頂，「覆蓋最廣、精度最粗——適合廣播式提醒『山腰有優惠』。」

「進村呢？」

「**Wi-Fi**。進到村裡就收得到。」

「靠近攤位——」

「**Beacon**。幾公尺內觸發。」

「最後結帳——」

「**NFC**。貼上去才扣款。」

布販恍然：「……我把四段混成了一段。」

費倫筆記：

> 常見偏差——**範圍錯配**：
> - 山腳大範圍廣播用 NFC → 客人接觸不到 → **範圍不對**。
> - 結帳瞬間用 GPS → 精度太粗，無法識別客人 → **精度不對**。
> - 每一段必須對應正確範圍，**精度要配情境**。

---

## 六、四階齊走

下午開拔。商隊在山腰用**GPS**對齊路徑，濃霧中沒迷路；進村前改用**Wi-Fi**點名集合；到攤位前以**Beacon**推播歡迎訊息；客人掏錢結帳那瞬**NFC**感應扣款。

領隊一路笑到合不攏嘴：「四段分明！原來這叫『四階』——」

修塔爾克難得挺起腰：「範圍從大到小——一路收進手心。」

費倫在旁輕輕補：「年代先後不等於範圍先後。考試很愛混這一點。」

芙莉蓮嚼著市集買的糖：「……都記好了？」

費倫點頭。

修塔爾克突然臉紅：「我、我剛剛說『收進手心』——是不是太中二？」

費倫憋笑：「師兄，蠻帥的。」

修塔爾克耳根又燒起來。

---

## 七、雙燈的提示

黃昏時，市集盡頭掛起兩盞燈籠——一盞紅、一盞藍。領隊解釋：「市集另有兩場測驗——一場考影像、一場考文字，考生要兩場同時上。」

費倫眼睛一亮：「同時上？」

「嗯。」領隊點頭，「一般考生只能一邊強、一邊弱。極少有人兩邊都旺。」

芙莉蓮瞇眼看那兩盞燈：「……費倫。」

「嗯？」

「明天——兩盞一起點。」

費倫微微一愣：「師父是說……多任務？」

「……兩個任務共用心像。」芙莉蓮吃掉最後一顆糖，「不是每顆糖都一樣甜——權重要自己調。」

修塔爾克在旁小聲：「一升一降……最常見了。」

費倫把筆記翻到新的一頁，只寫下一行：

> **多任務 — 損失權重**

山霧在山腰完全散盡。雙燈的光一前一後映進費倫眼底。

## 芙莉蓮的冒險筆記（S1E08）

| 項目 | 內容 |
|---|---|
| 英文全名 | **GPS** / **Wi-Fi** / **Beacon** / **NFC** |
| 中文意譯暱稱 | 定位羅盤四階（天・村・攤・手） |
| 年代 | GPS 1973 ／ Wi-Fi 1997 ／ **NFC 2004** ／ **Beacon 2013** |
| 所屬科目 | 科目一｜位置技術 |

**一句話口訣**
- **覆蓋範圍（大→小）**：GPS → Wi-Fi → Beacon → NFC。
- **精度（粗→細）**：同上順序。
- **行銷精度依情境選擇**——場景對應範圍。

**考試重點速記**
- **GPS**：全球衛星定位，最廣、最粗；適合城市／山域／大範圍廣播。
- **Wi-Fi**：建築／社區範圍；進村、進商場後才收得到。
- **Beacon**：店內／攤位前的藍牙訊標，數公尺觸發；近場行銷。
- **NFC**：接觸感應（數公分內），**必須貼上**才成立；結帳、票證。
- **易混題型**：年代順序（1973 → 1997 → **2004（NFC）** → **2013（Beacon）**）與範圍順序（GPS → Wi-Fi → Beacon → NFC）**不同向**。
- 常考「下列情境應選何種定位技術」；**場景對應範圍**為解題關鍵。

**心像崩潰對照**（本章為修塔爾克工程示範，非魔法咒紋）
- 大範圍廣播用 NFC → 客人走不到攤前就無效 → **範圍不對**。
- 結帳瞬間用 GPS → 精度太粗、無法識別客人 → **精度不對**。
- 正確：由粗入細——GPS 進山 → Wi-Fi 進村 → Beacon 到攤 → NFC 貼手。
