## S1E01｜〈萬法之祖〉— Bayes' Theorem（1763）— 芙莉蓮

- **章名（完整）**：`S1E01｜〈萬法之祖〉—— Bayes' Theorem（1763）`
- **地點／天氣**：北方雪原邊緣的廢棄修道院；初春雪未化，灰雲低壓。
- **委託／矛盾**：村長委託芙莉蓮一行調查「修道院藏書是否為禁咒」；費倫初次跟班，期待見識芙莉蓮真本事。費倫想「直接讀殘卷判斷」，芙莉蓮卻只想「先估事前機率」。
- **3 次對白交鋒概要**：
  1. 費倫主張「翻開書一看就知道」；芙莉蓮反問「不看書時，你猜這是禁咒的機率多少？」費倫答不出。
  2. 費倫翻到一頁咒紋，下結論「這就是禁咒」；芙莉蓮一句「分母是 B，不是全樣本」打斷。
  3. 費倫嘗試重算 P(禁咒|咒紋)，仍把 P(B) 漏掉；芙莉蓮示範心像構築，靜默發動。
- **首次發動描寫方向**：芙莉蓮閉眼，書頁上浮出極淡的條件機率咒紋（觀察視角由費倫描繪），片刻後芙莉蓮口語確認「……Bayes' Theorem。」
- **考試重點覆蓋清單**：
  - 公式 P(A|B) = P(B|A)×P(A)/P(B) — 由芙莉蓮口頭背誦並反問費倫。
  - 「Monte Carlo 估算條件機率用 A_and_B.sum()/B.sum()」— 芙莉蓮舉例：以三百次擲骰子模擬同條件，現場演算（伏筆 S1E09）。
  - 先驗、後驗、概似度三者區分 — 由費倫筆記整理。
- **伏筆銜接**：芙莉蓮提及「條件機率不只用骰子，也能用在『誤差三尺』的尺度選擇」，引出 S1E02。
- **上集伏筆引文**：（S1E01 為全書開場，無引文。）

## S1E02｜〈誤差三尺〉— MSE / MAE / RMSE（1809）— 芙莉蓮

- **章名**：`S1E02｜〈誤差三尺〉—— Mean Squared / Mean Absolute / Root Mean Squared Error（1809）`
- **地點／天氣**：返村途中的三岔路口酒館；春雨綿綿。
- **委託／矛盾**：村人請眾人估算「下季作物收成偏差」。費倫主張用平均誤差就好；芙莉蓮認為要看「離群豐凶年」要用 MSE 放大尾端。
- **3 次對白交鋒**：
  1. 費倫：「不是都取絕對值就好？」芙莉蓮：「絕對值對大誤差不夠敏感。」
  2. 費倫提出 MSE，但忘了單位差異；芙莉蓮：「平方後單位變成『斤²』，村長聽不懂。」→ 引出 RMSE。
  3. 費倫想用 RMSE 當作誤差三尺的「唯一答案」；芙莉蓮反駁「異常值多時 MAE 較穩健」。
- **首次發動方向**：芙莉蓮在地上排出三組石子模擬殘差，三尺光紋同時亮起，靜默吐出「……Mean Squared Error、Mean Absolute Error、Root Mean Squared Error。」
- **考試重點覆蓋清單**：
  - MSE 平方後放大大誤差 — 「平方放大尾巴」一句點破。
  - MAE 較穩健 — 對應「離群多用 MAE」台詞。
  - RMSE 單位與原始資料相同 — 「斤²／斤」對話演示。
- **伏筆銜接**：芙莉蓮提到「想看誤差是否是離群，下章我教你 Z-Score。」
- **上集伏筆引文**：「分母是 B，不是全樣本。」（S1E01 落幕句）

## S1E03｜〈離群之尺・時間之鳥・間隔之蛇〉— Z-Score / Poisson / Exponential / CDF・PDF / IQR — 芙莉蓮（5 技術合篇）

- **章名**：`S1E03｜〈離群之尺・時間之鳥・間隔之蛇〉—— Z-Score / Poisson / Exponential / CDF・PDF / IQR（1800s–1837／基礎）`
- **地點／天氣**：邊境驛站的鐘樓；暮色，晚風帶霜。
- **委託／矛盾**：驛長記錄「夜行馬車每小時通過數」異常，要求判斷是否真有怪物潛伏。費倫直接用平均；芙莉蓮提醒「先看分佈型態」。
- **3 次對白交鋒**：
  1. 費倫：「平均每小時 3 輛，今晚 8 輛太多！」芙莉蓮：「Z 大於 3 才算異常。」
  2. 費倫換用 Poisson 算機率；芙莉蓮反問「事件間隔的分佈呢？」 → Exponential 對偶登場。
  3. 費倫混淆 PDF 與 CDF；芙莉蓮：「CDF 是 PDF 的積分。」最後以 IQR 作 sanity check。
- **首次發動方向**：芙莉蓮指尖點出五道暈光（Z／PDF／CDF／Poisson／Exponential 同時啟動），口頭逐一確認真名。
- **考試重點覆蓋**：
  - Z = (X-μ)/σ；Z≥3 判異常。
  - Poisson 描述次數（離散）／ Exponential 描述間隔（連續），互為對偶。
  - CDF = ∫PDF dt。
  - IQR = Q3 - Q1，箱型圖判離群。
- **伏筆銜接**：芙莉蓮一句「分佈不對稱時，光看標準差會被偏態騙——下章再說。」
- **上集伏筆引文**：「想看誤差是不是離群，我下章教你 Z-Score。」

## S1E04｜〈偏移之問・相關之繩〉— Skewness / Pearson r — 芙莉蓮

- **章名**：`S1E04｜〈偏移之問・相關之繩〉—— Skewness / Pearson Correlation Coefficient（1895–1896）`
- **地點／天氣**：山谷田畝；正午曝日，旱象初顯。
- **委託／矛盾**：地主擔憂田畝產量與灌溉量「呈反比」。費倫直接畫散布圖；芙莉蓮先要求看分佈是否偏斜。
- **3 次對白交鋒**：
  1. 費倫：「相關係數 0.3，幾乎沒關係。」芙莉蓮：「先看尾巴。右偏會把 r 拉低。」
  2. 費倫嘗試剔除離群；芙莉蓮：「偏態 > 0 是右偏，尾巴往右。」
  3. 費倫重算 r，得 0.78，正相關；芙莉蓮一句「[-1, 1] 之間，1 完全正相關，-1 完全負相關。」
- **首次發動方向**：芙莉蓮在地上拉出一條微微震顫的光繩（r），田畝對應點亮——靜默吐出「……Pearson r。」
- **考試重點覆蓋**：
  - 偏態 < 0 左偏／> 0 右偏。
  - r 範圍 [-1, 1]；極值意義。
- **伏筆銜接**：芙莉蓮：「相關不等於獨立——下章 Chi-Square 才是判印。」
- **上集伏筆引文**：「分佈不對稱時，光看標準差會被偏態騙。」

## S1E05｜〈獨立判印・重疊之眼〉— Chi-Square / IoU — 芙莉蓮

- **章名**：`S1E05｜〈獨立判印・重疊之眼〉—— Chi-Square Test / Intersection over Union（1900–1901）`
- **地點／天氣**：邊境戰場遺跡；陰天，焦土未冷。
- **委託／矛盾**：學者要求判定「兩個族裔的劍痕花樣是否獨立」。費倫想用相關係數；芙莉蓮：「類別變數要用 χ²。」並順勢以兵器邊界框教 IoU。
- **3 次對白交鋒**：
  1. 費倫主張用 r；芙莉蓮：「r 是連續變數的，類別獨立性要 χ²。」
  2. 費倫框出兵器圖，框得太鬆；芙莉蓮：「IoU = 交集 ÷ 聯集，閾值越高精度要求越嚴。」
  3. 費倫設 IoU 閾值 0.3；芙莉蓮：「0.5 起跳，配 mAP 用。」
- **首次發動方向**：χ² 判印與 IoU 雙重光圈在劍痕上重疊，芙莉蓮口語確認真名。
- **考試重點覆蓋**：
  - χ² 用於類別變數獨立性／適合度。
  - IoU = 交集／聯集；閾值與 mAP 搭配。
- **伏筆銜接**：費倫：「下次能不能用一條軸把高維資料壓平？」（引 PCA）
- **上集伏筆引文**：「相關不等於獨立——下章 Chi-Square 才是判印。」

## S1E06｜〈軸線濃縮・解釋之冠〉— PCA（費倫首次主導）/ R²

- **章名**：`S1E06｜〈軸線濃縮・解釋之冠〉—— Principal Component Analysis / R²（1901–1921）`
- **地點／天氣**：王立圖書館地下書庫；昏黃燭火，書塵漂浮。
- **委託／矛盾**：館長委託壓縮龐大族譜資料以利檢索。費倫初次主導施法，緊張；芙莉蓮全程旁觀只丟一句指點。
- **3 次對白交鋒**：
  1. 費倫沒給 n_components；芙莉蓮：「不指定就保留全部，等於沒降維。」
  2. 費倫設 n=2，累積解釋變異 62%；芙莉蓮：「> 80% 才安全。」
  3. 費倫升 n=4，得 R² = 0.85；費倫誤說「準確率 85%」；芙莉蓮：「是『可解釋變異 85%』，不是準確率。」
- **首次發動方向**：費倫獨力完成心像，淡藍主成分軸自書頁浮出，吐出「……Principal Component Analysis。」（費倫首次成功主導）
- **考試重點覆蓋**：
  - 累積解釋變異 > 80% 可安全降維；未指定 n_components → 保留全部。
  - R² = 0.85 表示 85% 變異可解釋（非準確率）。
- **伏筆銜接**：芙莉蓮：「降維只是入口——真正考驗是『虛實檢定』。」
- **上集伏筆引文**：「能不能用一條軸把高維資料壓平？」

## S1E07｜〈虛實檢定・信賴之域〉— Hypothesis Testing / CI — 芙莉蓮

- **章名**：`S1E07｜〈虛實檢定・信賴之域〉—— Hypothesis Testing / Confidence Interval（1933–1937）`
- **地點／天氣**：王宮御前審議室；冷光，石柱迴聲。
- **委託／矛盾**：御醫宣稱「新療法明顯有效」；芙莉蓮被召作仲裁。費倫主張看樣本平均；芙莉蓮要求 p-value 與信賴區間。
- **3 次對白交鋒**：
  1. 費倫：「平均比舊療法高 12%，明顯有效。」芙莉蓮：「H₀ 是『沒差』，先看 p-value。」
  2. 費倫誤說「p < 0.05 表示 H₀ 為真的機率 < 5%」；芙莉蓮：「不是。p-value 不是 H₀ 為真的機率。」
  3. 費倫換看 95% CI；芙莉蓮：「信心水準愈高，區間愈寬；樣本愈大，區間愈窄。」最後芙莉蓮補刀：「永遠別說『接受 H₀』，只能說『無法拒絕 H₀』。」
- **首次發動方向**：芙莉蓮起手即發，雙重判印（H₀ / CI）懸於御醫病例上方，靜默確認真名。
- **考試重點覆蓋**：
  - H₀ 虛無假說定義；p-value ≠ H₀ 為真的機率。
  - 「無法拒絕 H₀」措辭。
  - 樣本數↑→區間↓；信心↑→區間↑。
- **伏筆銜接**：芙莉蓮：「拒絕／接受之外，還有『曲線下審判』——下章上場。」
- **上集伏筆引文**：「降維只是入口——真正考驗是『虛實檢定』。」

## S1E08｜〈曲線下審判・擠壓之刃〉— AUC-ROC / Sigmoid — 芙莉蓮＋費倫

- **章名**：`S1E08｜〈曲線下審判・擠壓之刃〉—— AUC-ROC / Sigmoid（1940s／1943）`
- **地點／天氣**：港口檢疫所；霧夜，海風帶腥。
- **委託／矛盾**：檢疫所要篩出 3% 染疫者，準確率高達 97% 卻全錯。費倫先用 Accuracy；芙莉蓮：「不平衡資料看 AUC-ROC。」順勢由費倫負責 Sigmoid 把 logit 擠壓進 (0, 1)。
- **3 次對白交鋒**：
  1. 費倫：「Accuracy 97%。」芙莉蓮：「全猜陰性也能 97%。」
  2. 費倫繪 ROC 曲線；芙莉蓮：「AUC = 0.5 隨機，= 1 完美，與門檻無關。」
  3. 費倫算機率前漏裝 Sigmoid；芙莉蓮：「Sigmoid 把分數擠進 0 到 1，二元分類用。」
- **首次發動方向**：芙莉蓮主咒（AUC-ROC），費倫副咒（Sigmoid）；ROC 弧光浮起後費倫補上 Sigmoid 將分數壓進 (0,1)，雙人各自吐真名。
- **考試重點覆蓋**：
  - AUC=0.5 隨機／=1 完美；不受不平衡影響、與門檻無關。
  - Sigmoid 輸出 (0,1)，二元分類用；多類用 Softmax（伏筆 S2）。
- **伏筆銜接**：芙莉蓮：「不平衡 + 高 Accuracy 是陷阱——這個之後還會回來。」（伏筆 S5E10 Accuracy Pitfall）
- **上集伏筆引文**：「拒絕／接受之外，還有『曲線下審判』。」

## S1E09｜〈隨機之瞳・連語之鎖・交叉熵刃〉— Monte Carlo / N-gram / Cross-Entropy

- **章名**：`S1E09｜〈隨機之瞳・連語之鎖・交叉熵刃〉—— Monte Carlo / N-gram / Cross-Entropy（1946–1948）`
- **地點／天氣**：篝火夜營；星空澄澈，空氣冷凝。
- **委託／矛盾**：旅商希望用古文殘卷預測下個字以還原全卷；費倫想直接套用機率，芙莉蓮先以 Monte Carlo 估條件機率，再交給費倫上 N-gram、CE。
- **3 次對白交鋒**：
  1. 芙莉蓮：「面對高度隨機性——蒙地卡羅。」費倫：「不是 K-means？」芙莉蓮：「不是。」
  2. 費倫設 N=2 連語；芙莉蓮：「N-gram 抓不到長距依賴。」
  3. 費倫以 MSE 算分類損失；芙莉蓮：「分類用 Cross-Entropy。選錯會不收斂。」
- **首次發動方向**：芙莉蓮主咒 Monte Carlo（隨機之瞳浮現）；費倫接手以 N-gram 連結光鎖、Cross-Entropy 切刃。三道真名依序吐出。
- **考試重點覆蓋**：
  - Monte Carlo 用於高度隨機（氣候）；非 K-means／SVR。
  - N-gram 僅靠固定長度前序詞，無長距依賴；不是「假設詞獨立」（那是 Naive Bayes）。
  - 分類用 CE，迴歸用 MSE；二元用 Binary CE。
- **伏筆銜接**：費倫筆記末頁畫了個方陣——「下章學矩陣判印？」
- **上集伏筆引文**：「不平衡 + 高 Accuracy 是陷阱。」

## S1E11｜〈下坡步法・散度之尺〉— SGD / KL Divergence

- **章名**：`S1E11｜〈下坡步法・散度之尺〉—— Stochastic Gradient Descent / KL Divergence（1951）`
- **地點／天氣**：山谷階地；薄霧，下行小徑。
- **委託／矛盾**：費倫的 PCA 模型過大，要找最低損失；芙莉蓮教 SGD「沿梯度下坡」；KL 用以量分佈差。
- **3 次對白交鋒**：
  1. 費倫想算全資料梯度；芙莉蓮：「Stochastic 就是隨機抽，便宜又快。」
  2. 費倫不知 momentum 來源；芙莉蓮：「SGD 沒內建 momentum——要手動加。」（伏筆 S4 Adam 對照）
  3. 費倫用 MSE 比兩個分佈；芙莉蓮：「分佈差用 KL Divergence；之後也用來偵測飄移。」（伏筆 S5E09 Drift）
- **首次發動方向**：費倫主咒 SGD（下坡光紋層層遞減）；芙莉蓮疊上 KL 之尺。各自吐出真名。
- **考試重點覆蓋**：
  - SGD 1951；無內建 Momentum，需手動加（與 Adam 比對）。
  - KL Divergence 衡量分佈差；可偵測 Data Drift（伏筆）。
- **伏筆銜接**：「明日我們進遺跡——三神判庭。」

## S1E13｜試煉Ⅱ〈古代總考〉— 季末大戰

- **章名**：`S1E13｜〈古代總考〉—— 季末大戰（1763–1951）`
- **整合對象清單**（取自 E01–E12，不含 E12）：
  - Bayes' Theorem（S1E01）
  - Principal Component Analysis（S1E06）
  - AUC-ROC（S1E08）
  - Cross-Entropy（S1E09）

  - Stochastic Gradient Descent（S1E11）
  - KL Divergence（S1E11）

- **四拍節奏**：
  1. **情報**：費倫先以 Bayes 計算「索菲斯為真神／為魔」事前事後機率。
