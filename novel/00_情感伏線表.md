# 情感伏線表（與技術依賴鏈獨立維護）

> **規則**：每季至少埋 1 條跨 ≥3 季的情感伏線；伏筆載體可為**道具／台詞／天氣／氣味／音樂／場景／動作習慣**，不必綁技術。
> **與 `00_十季時間軸總覽.md` 並行**：技術依賴記在總覽，情感伏線記在本檔。
> **§A.7 末條硬規**：情感伏線權重 ≥ 技術族譜回扣；非線性回扣（後章某動作觸發前季記憶）優於線性技術依賴展示。
> **與呼吸章關係**：F-001 以外所有跨季伏線均**首建立或主強化於呼吸章**（見 `00_十季時間軸總覽.md §4.2`）；技術章僅作回扣，不作首建立。

## §2 各條伏線詳細設計

### F-001｜芙莉蓮的「沒事」

- **載體**：純台詞；無道具實體
- **語感**：原作中芙莉蓮的慵懶口頭禪。前期語氣輕（推託、避答）；中期偶爾出現節奏異常（被打斷、停頓過長）；末期成為自我封閉訊號（迴避情感對話）
- **使用節律**：
  - **S1E01**（全書開場）：作為慵懶人格的標籤——費倫問「為什麼貝氏定理用先驗？」芙莉蓮答「沒事，先驗就那樣」
  - **S1E10**（靜默收束）：故學派老人離場後，費倫追問「妳認識他嗎？」「妳當年也是這群人之一嗎？」「那一朵花沒有人收回去嗎？」——芙莉蓮三聲「沒事」，第二次起句尾的拖比第一次長半拍；費倫未寫進筆記，只是記著
  - **S2E01**（假說檢定章）：費倫追問「妳真的會教我這條尺嗎？」——芙莉蓮答「沒事」，句尾的拖比 S1E01 那晚又拖長一點點；隨後芙莉蓮自己又開口「會教」，這是「沒事」首次被自我修正（節律維持期內的微小裂紋）
  - **S5E10**（靜默收束）：費倫問「妳為什麼不上前阻止那個旅人？」芙莉蓮答「沒事」，費倫進一步察覺異樣
  - **S8E13**（季末大戰）：戰後修塔爾克受傷，芙莉蓮答「沒事」——但伸手扶人的動作慢半拍
  - **S10E07**（呼吸章・考前夜）：費倫不再問了，只是把雨夜咖啡推到芙莉蓮面前；芙莉蓮卻自己開口「⋯⋯不是沒事」——**口頭禪的反向破封**
  - **S10E13**（季末大戰／全書終章）：在送行段，芙莉蓮終於完整說出對辛美爾的話——「我以為我沒事」
- **撰寫硬規**：**前期 4 季每季至少出現 1 次**（不勉強，自然帶出）；**S6–S9 階段每季 ≤1 次**（節制保留爆發力）；**S10 用兩次完成「準破封→完成破封」**

### F-002｜辛美爾的銀懷錶

- **載體**：道具（實體）＋場景（光線）
- **物件規格**：銀製懷錶；錶蓋內側刻褪色花押；僅在燭光／夕陽斜照下花押能被肉眼辨識
- **使用節律**：
  - **S2E05**（OCR 章）：首登場——芙莉蓮在掃讀辛美爾舊信時，懷錶從信件夾層滑出
  - **S2E06**（呼吸章）：放在火爐邊桌；芙莉蓮的回憶閃回（與辛美爾年輕時對話）
  - **S4E01／E03**（雙鑰／HE 章）：辛美爾的勇者判例提及——「當年我們約好不可開啟對方的訊息，這只懷錶是另一把鑰匙」
  - **S4E05**（呼吸章・夜雨蠟印）：燭光下花押首次顯影；費倫第一次看見刻字
  - **S4E12**（季末大戰）：費倫接過懷錶（傳承動作）
  - **S9E06**（Transformer 章）：費倫在拆解多頭注意力時，懷錶滴答聲與她的內心節拍同步——成為「並行思考的節奏錨」
  - **S10E07**（呼吸章）：四角共聚之夜，懷錶最後一次擺上桌
  - **S10E11**（憲法式 AI 章）：辛美爾的回憶說「對齊不是控制，是繼承」——懷錶花押在說話間被夕陽斜照顯影
- **撰寫硬規**：花押**僅在燭光／夕陽**下顯影；不可在白晝或魔法光下顯影（保留神秘感與儀式感）；費倫接過懷錶後，從 S5 開始她隨身攜帶但不主動提及，成為**沉默的繼承證據**

### F-003｜雨夜咖啡的氣味

- **載體**：氣味（咖啡）＋天氣（雨）；後期疊加木屑（S7）
- **使用節律**：
  - **S5E04**（呼吸章）：雪原邊小屋，倫泡咖啡，雨打窗——氣味首建立
  - **S7E05**（呼吸章）：工坊內，咖啡氣味與木屑氣味疊加——**第二層氣味**形成
  - **S9E07**（呼吸章・車站）：費倫在大戰前夜聞到一杯陌生人的咖啡，**情感觸發**——成為鎮定錨點
  - **S10E07**（呼吸章・考前夜）：芙莉蓮親手泡咖啡（第一次），氣味第三次回收
  - **S10E13**（季末大戰）：大戰高潮前的最後一吸——費倫將鼻尖湊近自己手心（最後一個鎮定動作）
- **撰寫硬規**：氣味描述**避免直白「咖啡」二字反覆出現**——可用「焦褐」「濕暖」「隔著雨打窗的苦韻」等替代詞；S7 之後**必同時帶木屑或紙屑氣味**（雙層氣味才能觸發費倫情感）

### F-004｜修塔爾克挑魚刺

- **載體**：動作習慣
- **動作規格**：修塔爾克吃魚時必先用筷尖／刀尖按壓魚肉，**沿側線一根一根挑刺**——動作慢、專注、不容打擾
- **使用節律**：
  - **S3E05**（呼吸章）：首登場——市集午後鹽烤魚；費倫驚訝這位戰士竟有此細活
  - **S6E06**（呼吸章・縫合靜默）：縫合自己傷口時，**動作節奏與挑魚刺一致**（隱性回扣）
  - **S8E08**（K8s 章）：修塔爾克調度容器的動作被費倫描述為「像在挑魚刺」——首次明寫的隱喻
  - **S10E10**（AI Agent / Function Calling / MCP 章）：**正面收束**——費倫向芙莉蓮解釋 Function Calling 時，舉例說「修塔爾克挑魚刺：要哪根工具，他比我們都清楚」；修塔爾克難得地紅了一下臉
- **撰寫硬規**：S3–S7 期間**僅作動作描述，不寫角色解釋意義**；意義揭示集中於 S10E10 一次完成；S8 的隱喻是費倫單方面類比（修塔爾克未察覺）

### F-005｜繞路三天去看的花

- **載體**：道具（花）＋場景（路徑記憶）
- **物件規格**：一朵不知名的小花；芙莉蓮某次旅程中**為了看一眼這朵花繞路三天**——當時眾人不解，芙莉蓮無解釋
- **使用節律**：
  - **S1E04**（呼吸章）：費倫在整理芙莉蓮舊筆記時，發現一張壓花標本——首次浮現；芙莉蓮末段一句「以前，繞路三天去看過。」
  - **S1E05**（偏態／Pearson r 章）：觀測小屋桌上同枝小花隨風搖動，無對白——同章僅作場景過渡
  - **S1E09**（分佈四神重識章）：燭火夜讀末，雨夜窗外同枝再動——隱性回扣，費倫未抬頭
  - **S1E10**（靜默收束）：故學派老人離場時，將一朵壓花擱於石屋門檻；芙莉蓮看了一眼便走——花從筆記走向實體門檻
  - **S2E13**（季末大戰）：戰場上一束乾燥花從敵人遺物中掉出（非同一朵，但同色）——**錯位重逢**
  - **S5E10**（靜默收束）：雪原邊界，同色花瓣被風吹過——芙莉蓮看了一眼就走（呼應「沒事」第一次節奏異常）
  - **S8E07**（呼吸章・流星）：流星軌跡與花的搖擺被費倫描述為「同一條線」
  - **S8E13**（季末大戰）：大戰煙塵中飄出一瓣——芙莉蓮停手 0.5 拍
  - **S10E13**（全書終章・送行段）：考場前夜，**在芙莉蓮記憶投影中再開**——費倫第一次看見完整的那朵花
- **撰寫硬規**：**S1–S9 期間芙莉蓮從不解釋為何繞路**；S10E13 也不直白解釋——僅以投影、無對話收束（避免說教）；象徵「無用之路」的價值，**對照 iPAS 考試的「目的導向」形成情感反差**

### F-006｜修塔爾克的縫合線繩（任務 0-4 新增）

- **載體**：道具（實體線繩）＋動作（縫合）
- **物件規格**：修塔爾克隨身攜帶的多股麻線——**三股一束、可拆可合**；用於緊急縫合傷口、綁固武器、串連零件；末端有戰士分隊的舊染色標記
- **設計理由**（為何此季新增此伏線）：S6（卷積黎明）開始進入神經網路與集成模型時代，「**多股串接成一**」是 Random Forest（多樹投票）／ResNet（跳接串接）／MCP（協定串接）的共同肉身原型——需要一個**戰士可攜帶的物質載體**作為跨季隱喻錨。修塔爾克本就承擔「戰士隊形比擬工程」的角色，線繩順其角色分工，且與 F-004 挑魚刺形成**雙動作組合**（精細＋串接，缺一不可）。
- **使用節律**：
  - **S6E06**（呼吸章・縫合靜默）：**首建立**——修塔爾克自縫傷口；費倫看見三股線繩末端的舊染色，沒問是什麼
  - **S6E07**（Random Forest 章）：當天晚上倫翻書時喃喃「多樹投票，像線繩多股一束」——隱性回扣
  - **S6E13**（季末大戰）：費倫接手縫合修塔爾克的另一道傷——**動作傳承**（與 F-002 懷錶傳承形成雙傳承呼應）
  - **S9E01**（ResNet 章）：費倫推導跳接架構時，桌上放著一束線繩；她說「梯度跨層走，像線繩跨股串」——**首次明寫對映**
  - **S10E10**（MCP 章）：修塔爾克解釋協定時，將線繩三股拆開展示——「規格化的串接，誰來串都一樣」
  - **S10E07**（呼吸章・考前夜・終回收）：**三條舊線繩同時擺於桌上**——一條來自 S6（修塔爾克）、一條來自 S6E13（費倫接手過）、一條來自 S10E10（MCP 章拆解後重綁）；四角無對話注視，由芙莉蓮打破靜默：「⋯⋯不是沒事」（與 F-001 同步爆發）
- **撰寫硬規**：
  - 線繩末端的**舊染色標記**從不被解釋——保留神秘感（暗示修塔爾克過去的戰士分隊）
  - **三股不可寫成兩股或四股**——三股一束是肉身規格，對應 Random Forest 三投票、ResNet 三層跳接、MCP 三協定（請求／回應／心跳）的隱喻骨架
  - S6→S9 之間**僅作物件出現，不寫對映解釋**；對映集中於 S9E01 與 S10E10 兩次明寫
  - **與 F-004 挑魚刺的雙動作組合**：S6E06 縫合靜默章內，修塔爾克的縫合動作節奏應與 S3E05 挑魚刺一致（隱性自我回扣）
- **與技術的關聯**（強）：
  - Random Forest（多樹集成投票）→ S6E07
  - Bagging / Boosting / Stacking（多模型串接）→ S6E09
  - ResNet（殘差跳接）→ S9E01
  - MCP（標準化協定串接）→ S10E10
  - 隱性對映：seq2seq 編碼-解碼串接、Transformer 多頭並行、AI Agent 工具鏈

## §3 維護規則

1. 寫完任一章，若該章埋下／回收伏筆，於 §1 表格更新該列「強化／回收節點」欄。
2. 「載體類型」限：道具／台詞／天氣／氣味／音樂／場景／動作習慣。
3. 「與技術的關聯」可填「無」——這是本表的特色，刻意不依賴技術鏈。
4. 跨度 < 3 季的伏線可記，但**不計入「每季 ≥1 條」配額**。
5. 伏線回收後不刪除，標註回收章即可，作為全書統讀檢查依據。
6. **新增伏線前**：須檢查現有 6 條是否已能承擔——避免伏線過載（讀者記憶超荷）；新增後須立即更新 §1 表格與 §2 詳細設計。
7. **跨季伏線首建立優先呼吸章**（§A.7 末條配套硬規）；技術章僅作回扣與強化。

## §4 撰寫時的取用優先級

> 若一章可同時放入多條伏線動作，依下列優先順序取用（避免單章伏線過載）：

1. **本季配額未滿的伏線優先**（每季 ≥1 條跨 ≥3 季伏線）
2. **回收 > 強化 > 首建立**（保證情感閉環）
3. **技術章內**：以**動作／氣味／道具**為載體（避免占用對白）；**呼吸章內**：可用台詞展開
4. **同章伏線數 ≤2 條**（除 S10E07／S10E13 全書收束章可達 4–6 條）
5. **試煉章與大戰章**：以伏線回收為情感主軸（§A.5 step 6「角色檢視自己」），技術整合為次
